ZEBRA is a research project that introduces a novel approach to commonsense question answering using zero-shot example-based retrieval augmentation. The system demonstrates improved performance across multiple benchmarks by effectively leveraging retrieved examples to enhance language model reasoning capabilities.

**Key Features:**
- Zero-shot learning approach
- Example-based retrieval
- Enhanced commonsense reasoning
- State-of-the-art performance on multiple benchmarks

**Publication:** EMNLP 2024
