CroCoAlign is a cross-lingual sentence alignment system designed for long texts. The system uses a fully-neural, context-aware approach that outperforms traditional alignment methods by considering broader context and semantic relationships across languages.

**Key Features:**
- Fully-neural architecture
- Context-aware alignment
- Support for long text documents
- Cross-lingual capabilities

**Publication:** EACL 2024

**Applications:**
- Machine translation
- Cross-lingual information retrieval
- Multilingual document processing
